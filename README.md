# react-native-yappasdk

Yappa SDK - Audio & Video Commenting tool

## Installation

```sh
npm install react-native-yappasdk
```

## Usage

```js
import Yappasdk from "react-native-yappasdk";

// ...

const result = await Yappasdk.multiply(3, 7);
```
